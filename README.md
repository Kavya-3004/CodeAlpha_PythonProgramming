# Basic Chatbot

A simple rule-based chatbot built using Python.

## Features
- Responds to greetings
- Answers basic questions
- Runs continuously until the user types "bye"
- Handles unknown inputs

## Technologies Used
- Python 3

## How to Run

```bash
python chatbot.py

## Sample Inputs
- hello
- how are you
- what is your name
- thank you
- good morning
- good night
- bye
